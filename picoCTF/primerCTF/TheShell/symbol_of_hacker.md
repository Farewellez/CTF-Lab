## 2.1 - Symbol of the Hacker

> "Layar hitam, kursor berkedip, dan teks misterius — itulah simbol klasik hacker."  

### 🖥️ Apa Itu Shell?

- Antarmuka teks antara pengguna dan sistem (alias: terminal, bash, command prompt, PowerShell)
- Digunakan untuk mengendalikan sistem operasi dengan perintah langsung

### 💥 Kenapa Shell Itu Kuat?

- **Tanpa konfirmasi**: menghapus file langsung terjadi
- **Akses penuh** ke direktori dan file sistem
- **Tanpa undo** – jadi harus hati-hati

### 🧠 Insight

- Shell bukan hanya alat teknis, tapi **ikon budaya hacker**
- Pengguna biasa memakai GUI (graphical interface), hacker dan power user memakai **shell**
- Belajar shell = membuka akses ke kekuatan sistem yang sesungguhnya

### 🚩 Dalam CTF

- Digunakan untuk menavigasi sistem, mengeksploitasi program, membaca flag
- Kemampuan shell yang kuat bisa **menjadi pedang bermata dua** jika tak digunakan dengan bijak
